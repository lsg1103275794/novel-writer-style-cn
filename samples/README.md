# 样本文件目录

这个目录用于存放要学习风格的小说文本文件。

## 目录结构建议

```
samples/
├── author-a/          # 作者A的作品
│   ├── novel1.txt
│   ├── novel2.txt
│   └── novel3.txt
├── author-b/          # 作者B的作品
│   ├── work1.md
│   └── work2.md
└── reference/         # 参考作品
    └── target-style.txt
```

## 文件要求

1. **格式**: 支持 .txt 和 .md 格式
2. **编码**: 使用 UTF-8 编码
3. **内容**: 纯文本，避免包含目录、页码等非正文内容
4. **长度**: 建议单个文件至少 10,000 字，总计 50,000 字以上

## 使用方法

1. 将要学习的小说文本放入对应目录
2. 使用 `/style-analyze` 分析风格
3. 使用 `/style-learn` 学习风格
4. 使用 `/write-styled` 进行风格化创作

详细使用方法请参考: plugins/style-learning/examples/usage-example.md

## 注意事项

⚠️ **版权提醒**: 请确保分析的作品仅用于个人学习，尊重原作者版权。

⚠️ **文件质量**: 样本质量直接影响学习效果，请选择高质量的文本文件。

⚠️ **样本数量**: 建议每个作者至少提供3-5万字的样本，多部作品效果更佳。
