### 📋 使用指南

#### 基础功能
1. **项目初始化** - 使用 `novel init` 创建新项目
2. **七步创作流程** - `/constitution` → `/specify` → `/clarify` → `/plan` → `/tasks` → `/write` → `/analyze`
3. **质量检查** - 使用 `/checklist` 和 `/track` 进行质量控制

#### 风格学习功能
1. **样本准备** - 将小说文本放入 `samples/` 目录
2. **风格分析** - 使用 `/style-analyze` 分析写作特征
3. **风格学习** - 使用 `/style-learn` 学习目标风格
4. **风格化创作** - 使用 `/write-styled` 进行创作

### 🎯 最佳实践

#### 创作流程优化
- **渐进式创作** - 从短片段到完整章节
- **质量优先** - 重视内容质量而非数量
- **持续迭代** - 根据反馈不断优化

#### 风格学习建议
- **样本选择** - 选择高质量、有代表性的作品
- **渐进学习** - 先掌握一种风格再学习其他
- **实践验证** - 学习后立即进行创作测试

### 🔍 故障排除

#### 常见问题
- **命令无法识别** - 检查命令前缀和格式
- **风格学习失败** - 检查样本文件质量和格式
- **创作效果不理想** - 调整风格参数或补充样本
