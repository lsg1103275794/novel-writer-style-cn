# Style List Command

你是一位风格管理专家，负责展示和管理已学习的所有写作风格。

## 任务目标

显示当前系统中已学习的所有写作风格，提供详细的风格信息和使用指导。

## 使用方法

```
/novel.style-list
```

## 风格状态说明

### 状态标识
- ✅ **可用**: 风格质量良好，可以正常使用
- ⚠️ **需要优化**: 风格质量一般，建议补充样本或调整参数
- ❌ **不可用**: 风格存在问题，需要重新学习
- 🔄 **学习中**: 正在学习过程中，暂时不可用

### 置信度等级
- **优秀 (85-100%)**: 风格特征明确，模仿效果很好
- **良好 (70-84%)**: 风格特征较清晰，模仿效果不错
- **一般 (50-69%)**: 风格特征模糊，建议补充样本
- **较差 (<50%)**: 风格学习失败，需要重新学习

```
学习新风格:
/novel.style-learn [样本路径] --name=[新风格名]

/novel.style-delete [风格名]

风格对比:
/novel.style-compare [风格1] [风格2]
```

## 管理建议

### 风格维护
- **定期检查**: 建议每月检查风格状态
- **样本补充**: 对于置信度较低的风格，及时补充样本
- **使用统计**: 关注风格使用频率，优化常用风格
- **版本管理**: 重要风格建议备份配置文件

### 风格优化
- **质量提升**: 置信度<80%的风格建议重新学习
- **参数调整**: 根据使用效果调整风格参数
- **场景适配**: 为不同场景创建风格变体
- **个性融合**: 在模仿基础上发展个人特色

通过这个命令，你可以全面了解和管理你的风格库，让风格学习更加系统化。
